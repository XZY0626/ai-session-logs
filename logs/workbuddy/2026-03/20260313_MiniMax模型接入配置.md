## 飞书端使用说明

在飞书中 @OpenClaw 机器人，发送以下命令可切换到 MiniMax 模型：

```
/model minimax/MiniMax-Text-01       # 旗舰通用，百万上下文
/model minimax/MiniMax-M1            # 推理旗舰，百万上下文
/model minimax/MiniMax-M2.5          # 最新旗舰
/model minimax/MiniMax-M2.5-highspeed # 快速版
```

或使用 alias 简写：
```
/model minimax-text
/model minimax-m1
/model minimax
/model minimax-fast
```
