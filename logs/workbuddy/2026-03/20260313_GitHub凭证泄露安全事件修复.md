### 1. 立即修改虚拟机密码（最高优先级）

**在VMware虚拟机Ubuntu中执行：**
```bash
passwd xzy0626
# 输入新密码（建议使用强密码，至少12位，包含大小写字母、数字、特殊字符）
```

### 2. 设置VM_PASSWORD环境变量（修改后）

**在Windows宿主机 PowerShell 中（当次会话）：**
```powershell
$env:VM_PASSWORD = "你的新虚拟机密码"
```

**永久设置（系统环境变量）：**
```
控制面板 → 系统 → 高级系统设置 → 环境变量

```
